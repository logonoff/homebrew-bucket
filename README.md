# homebrew-bucket

Scoop bucket / Homebrew tap for my stuff

## Homebrew

```zsh
brew tap logonoff/bucket
```

### Available Casks

| Cask | Source |
| ---- | ------ |
| `superopt` | [logonoff/superopt](https://github.com/logonoff/superopt) |

## Scoop

```powershell
scoop bucket add logonoff https://github.com/logonoff/homebrew-bucket
```
